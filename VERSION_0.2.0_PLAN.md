## § 1 详细任务单

- [x] **P1-2: 仪表盘 UI 开发**
  - 新建 `DashboardView.vue`。
  - 实现任务卡片列表：支持实时进度展示和“一键终止”功能。

### P2: 自动化版本更新
- [ ] **P2-1: 更新体系配置**
  - 配置 `tauri-plugin-updater`。

### P3: 视频解析与知识提取 (API 版)
- [ ] **P3-1: 视频处理脚本**
  - 新增 `scripts/video_ingestor.py`。
  - 支持通过链接下载视频并分离音频 (FFmpeg)。
- [ ] **P3-2: 集成火山引擎 ASR**
  - 对接 API 接口，将音频流上传并获取转录文本。

## § 2 关键决策
1. **STT 方案**：放弃本地运行 Whisper，统一采用 API 方式（首选火山引擎 ASR）。
2. **进程管理**：Rust 端需实现对子进程的强引用管理，防止主程序崩溃后产生僵尸进程。
